# WV-Components

Deze repository bevat een project dat gebruik maakt van React, Vite, TailwindCSS en Storybook. Het stelt je in staat om componenten te ontwikkelen en te testen in Storybook, de applicatie voor productie te bouwen en de code te linten voor kwaliteit.

Gebruik de volgende commando's om het project op te zetten en te starten:

```bash
$ npm install  # Installeer alles wat nodig is voor het project
$ npm run storybook  # Start Storybook op http://localhost:6006
$ npm run lint  # Voer ESLint uit om codeproblemen te vinden
$ npm run build  # Bouw de productieversie van de applicatie
```
